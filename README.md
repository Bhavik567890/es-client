# Frontend (React + TypeScript) - Blog App

A simple frontend for the Blog Application built using **React**, **TypeScript**, **Redux Toolkit**, **React Router**, and **SCSS**.

##  Features

* Create, edit, delete blog posts
* View blog list and blog details
* Word highlight feature across list & details
* State management using Redux Toolkit
* Fully typed with TypeScript

## Tech Stack

* React + TypeScript
* Redux Toolkit
* React Router
* SCSS

##  Run the Project

```bash
npm install
npm run dev
```
